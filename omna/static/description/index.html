Frequently Asked Questions

What's the requirements to use this connector?

We support Odoo V13 and above. Also, a paid Cenit IO (www.cenit.io) subscription plan is required, starting from USD$19/month. 

What is Cenit IO?

Cenit IO is an open-source integration platform as a service. OMNA API is a paid feature built on Cenit IO to enable merchants to quickly and affordably connect their Odoo ERP to various sales channels. Cenit IO also offers custom integration for Odoo users, please drop us an email at support@omna.io to find out more.
